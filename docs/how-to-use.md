# spot-diggz How To Use

## 何のためのサービスか

spot-diggzは、スケートパークの一覧や地図を提供するだけのサービスではありません。

「今日は基礎を練習したい」「仕事のあとに短時間だけ滑りたい」「久しぶりなので無理なく滑りたい」といった、その日の条件から、実際に行く施設を決めるためのサービスです。

施設の営業時間、休場、設備、初心者向け情報、料金、アクセスなどを確認し、現在の条件に合う候補を最大3件まで理由付きで表示します。

## 使い方

### 1. 本番URLを開く

[https://spotdiggz.vercel.app](https://spotdiggz.vercel.app) を開きます。

### 2. まずはワンクリックで探す

入力を細かく決めていない場合は、次のいずれかを使います。

- **おまかせで今から滑る**: 現在の基本条件で、今から利用しやすい候補を探す
- **気軽に**: 近さと使いやすさを重視する
- **じっくり**: 練習したい目的との相性を重視する
- **挑戦する**: ランプ、ボウル、バンクなどの設備を重視する

### 3. 必要に応じて条件を変更する

「条件を変更」から、次の項目を選択できます。

- 検索位置: 大阪駅などの代表地点、駅名・住所、現在地
- 練習したいこと: 基礎、ストリート、ランプ・ボウル
- 気分: じっくり、気軽に、挑戦
- レベル: 初心者、久しぶり、中級
- 使える時間: 1〜4時間
- 交通手段: 公共交通、車、自転車、徒歩

現在地を使う場合は、ブラウザの位置情報許可が必要です。位置情報を許可しなくても、代表地点や駅名を使って検索できます。

現在地、経路、公式情報、訂正報告などの操作は、用途を表すアイコンと短い表示名で示します。アイコンだけで意味が分からない場合は、表示名を確認してください。

### 4. 結果を比較して行き先を決める

候補カードでは、次を確認できます。

- おすすめ理由
- 通常営業時間と当日の利用可能性
- 料金、設備、初心者向け情報
- 出発地点からの移動時間の目安
- 公式情報の出典と確認時期
- 外部ナビゲーションへのリンク
- 施設の公式SNSプロフィールへのリンク（登録されている場合）
- 手動で確認したYouTube動画（登録されている場合、施設ごとに最大1件）

営業時間や休場情報は、最終的に施設の公式ページでも確認してください。施設の貸切、天候、混雑など、公開情報だけでは判断できない状況は保証されません。

動画は「動画を表示」を選択した後にYouTubeから読み込みます。自動再生はしません。動画を表示できない場合は、通常のYouTubeリンクを開けます。動画は撮影時点の様子であり、現在の設備、混雑、利用可否、安全性を保証するものではありません。

SNSは施設の公式性を確認できたプロフィールだけを外部リンクとして表示します。spot-diggz内で投稿、ハッシュタグ、フィードは表示・収集しません。

### 5. 情報の誤りを報告する

施設カードの「情報の誤りを報告」から、営業時間、料金、ルール、休場、アクセスなどの訂正を送信できます。施設や自治体の公式ページがある場合は、根拠URLを添えてください。

連絡先メールアドレスは任意です。入力する場合は、利用目的と最長90日間の保持・削除に同意してください。訂正内容や根拠URLに、正確な現在地や不要な個人情報を入力しないでください。

## 情報の範囲と注意点

- 大阪府、兵庫県、和歌山県、奈良県、徳島県の施設を対象にしています。
- 施設カタログは公式情報を確認したデータを使用しています。
- 営業時間などの動的情報は30日、安定情報は180日を目安に更新期限を管理しています。
- 一般利用の予定を確認できない施設は、一覧には表示されても推薦対象から除外しています。
- Google/Apple Maps、SKEPA等の第三者サイトやSNSから、画像、動画、投稿を自動収集・保存・再配信していません。
- 動画は手動で選定したYouTubeへの外部連携です。動画表示を選択するとYouTubeへ通信されるため、利用前にYouTubeの規約とprivacy表示を確認してください。
- 現在の本番環境ではGoogle Maps連携を有効化していないため、移動時間は直線距離から算出した目安です。任意の住所検索は利用できません。

検索位置や検索条件は、spot-diggzに永続保存しません。Google連携を有効化した環境では、経路計算や地点検索のために位置情報・検索文字列がGoogleへ送信される場合があります。
