# 大阪都市圏の施設候補台帳

- Status: Candidate inventory
- Checked at: 2026-07-14
- Related issue: GitHub Issue #278
- Next issue: GitHub Issue #280

## 1. 目的

大阪市・堺市にいる初心者が、公共交通または自動車で概ね60分以内に到達できる可能性があるスケートボード施設を、MVPデータ作成前の候補として管理する。

この台帳への登録は掲載確定を意味しない。Issue #280で一次情報、移動時間、初心者適性、利用規約を確認し、掲載条件を満たした施設だけを検証済みカタログへ移す。

## 2. 候補の発見方法

候補は次の順序で発見した。

1. 大阪府「アーバンスポーツ施設について」の施設一覧
2. 自治体、公園指定管理者、施設運営者の公式ページ
3. AJSA全国スケートパーク情報
4. 施設運営者の公式店舗ページ

大阪府施設一覧とAJSA一覧は候補発見に使う。公開対象へ採用する根拠は、各自治体または施設運営者の一次情報とする。

## 3. 代表起点と60分圏

初期検証では次の4地点を代表起点とする。

- JR大阪駅
- なんば駅
- 南海堺駅
- なかもず駅

施設は、いずれかの代表起点から公共交通または自動車で概ね60分以内に到達できる場合に候補として残す。最終的な推薦では代表起点ではなく、利用者が入力した出発地点、出発時刻、交通手段から判定する。

## 5. 掲載条件

次をすべて満たした施設だけを検証済みカタログへ移す。

- 施設運営者または自治体の一次情報でスケートボード利用が確認できる
- 住所と位置を確認できる
- 営業時間または利用可能時間を確認できる
- 料金、登録、予約、防具、年齢等の主要ルールを確認できる
- 初心者向けのフラットエリア、スクール、レンタル等の根拠、または初心者不向きである根拠を確認できる
- 屋内外、照明、路面、主要セクションを確認できる
- 代表起点からの移動時間を交通手段別に確認できる
- 情報源URLと確認日を保存できる
- 掲載に使用する文章・画像・ロゴが第三者の権利を侵害しない

## 6. 検証結果の状態

- candidate: 候補発見のみ
- verifying: 一次情報を確認中
- verified: 必須属性を確認済み
- stale: 鮮度目標を超過
- excluded: 60分圏外、閉鎖、初心者向け判断不能、利用可否不明等により除外
